## Requisitos
Com base no back-end e nos detalhes fornecidos do projeto, crie o front-end, utilizando:

- React
- Vim
- CSS
- TypeScript
- Fetch

Para features que ainda não foram desenvolvidas no back-end, gerar mocks no front-end para o funcionamento
